# Minions Contacts

People, organizations, and contact channels

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-contacts/sdk`) — Core library for Node.js
- **Python SDK** (`minions-contacts`) — Python bindings
- **CLI Tool** (`@minions-contacts/cli`) — Command-line interface
- **Web App** — Interactive playground at [contacts.minions.wtf](https://contacts.minions.wtf)
