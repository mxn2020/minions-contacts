---
title: Installation
description: Install Minions Contacts packages
---

## TypeScript / Node.js

```bash
npm install @minions-contacts/sdk minions-sdk
```

## Python

```bash
pip install minions-contacts
```

## CLI

```bash
npm install -g @minions-contacts/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
